ğŸ§  Prompt Engineering Logic
The prompt system in this task uses the PAS (Problem-Agitation-Solution) framework:

The Hook: Grabs attention in the first 3 seconds by addressing a relatable pain point (e.g., "I can't focus at home").

The Agitation: Highlights the frustration of typical noisy environments.

The Solution: Introduces The Daily Grind as the ultimate escape.

The CTA: A low-friction call to action (e.g., "Link in bio to book a desk").
